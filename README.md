# 🚗 Project 2: Car Price Prediction

## Objective

To develop a machine learning model capable of predicting car selling prices using historical vehicle data.

## Workflow

- Data Cleaning & Preprocessing
- Feature Selection
- Train-Test Split
- Linear Regression
- Model Evaluation

## Model Performance
